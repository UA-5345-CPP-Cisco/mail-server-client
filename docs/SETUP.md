# Setup guide for maintainers and contributors

This guide provides a step-by-step instruction for preparing your environment for project development.

In case of errors or specific moments please open an issue with description of problem.

## Prerequisites 

- C++17
- CMake >= 3.20
- Ninja
- clang-format
- Qt >= 6.10
- OpenSSL
- SQLite3

## Dependencies installation

### MacOS

```
xcode-select --install

brew install cmake ninja qt clang-format
```
